Telegram Bot для рассылки

Это веб-интерфейс для управления Telegram ботом рассылки. Бот позволяет эффективно управлять массовыми рассылками в Telegram.

Основные возможности:

  • Авторизация через Telegram
  • Управление списками рассылки
  • Создание и отправка сообщений
  • Планирование рассылок
  • Статистика отправки

⚠️ Примечание: Это только демонстрационная страница. Для полноценной работы с ботом требуется установка на сервер.
